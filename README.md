# stylelint-config-rscss-base

> The stylelint config for rscss.

## Install

```bash
npm install --save-dev stylelint stylelint-config-rscss-base
```

## Usage

Set your `.stylelintrc` config to:

```json
{
  "extends": "stylelint-config-rscss-base"
}
```
